# Sistema de Control PID basado en IoT
## Descripción

Este proyecto implementa un sistema de control PID utilizando una arquitectura IoT basada en ESP32 y Raspberry Pi, permitiendo monitoreo y control en tiempo real mediante comunicación en red.

### Arquitectura del sistema
ESP32 para adquisición de datos y control de actuadores
Raspberry Pi para procesamiento y comunicación
Python para la lógica de control
Comunicación basada en sockets para transmisión en tiempo real

### Características
Control PID en tiempo real
Monitoreo y control remoto
Comunicación inalámbrica entre dispositivos
Integración de sistemas embebidos

### Tecnologías utilizadas
ESP32
Raspberry Pi
Python
Sockets

### Resultados
Respuesta estable del sistema ante diferentes condiciones
Comunicación confiable en tiempo real
